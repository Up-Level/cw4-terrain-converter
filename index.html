Creeper World 4 Terrain Converter

This will convert a heightmap generated by a script into a two-dimensional array that can be referenced by 4rpl directly.

To obtain the heightmap:

  • Close and reopen the map
  • Create a console script with this code: GetMapSize ->sizeZ ->sizeX <-sizeX 1 - ->sizeX <-sizeZ 1 - ->sizeZ CreateList ->heightmap <-sizeZ 0 do <-sizeX 0 do GetTerrain(i j) loop List <-heightmap swap AppendToList loop print(<-heightmap)
  • Run the script and navigate to the game files, usually found in Documents/my games/creeperworld4
  • Upload the file RPL.txt
